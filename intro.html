# 关于我

# 写博客的目的

技术这条路没有捷径，唯有不断学习和实践才能走得更远。建这个博客的初衷很简单：

  • 记录学习过程：把学到的技术点整理成文章，加深理解，也方便日后回顾
  • 探索技术边界：尝试不同的技术栈和工具，找到适合自己的方向
  • 发展兴趣爱好：编程不只是工作，也是兴趣，把兴趣变成专业
  • 分享交流：如果能帮助到其他人，那是最好的

# 技术理念

我相信技术是为人服务的，不是为了炫技。不管是写代码、做项目还是写博客，都应该：

  • 解决实际问题
  • 保持简单清晰
  • 持续改进优化

# 更新计划

计划每周更新一篇技术文章，内容涵盖：

  • 编程语言和框架
  • 物联网相关技术
  • 实际项目经验
  • 学习心得体会

当然，计划赶不上变化，有时候忙起来可能断更，但会尽量保持节奏。
