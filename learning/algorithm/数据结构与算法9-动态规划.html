                        数据结构与算法-动态规划

                        Kevin2025/12/26大约 13 分钟数据结构与算法学习数据结构与算法

                        动态规划是解决优化问题的利器，掌握动态规划能让你解决许多看似困难的问题

                        前言

                        动态规划（Dynamic Programming，简称DP）是一种通过把原问题分解为相对简单的子问题的方式求解复杂问题的方法。

                        核心应用：

                        • 最优化问题：背包问题、最短路径
                        • 计数问题：路径计数、组合计数
                        • 判定问题：字符串匹配、子序列问题

                        在本文中，我们将深入探讨：

                        1. 动态规划的核心思想
                        2. 动态规划的解题步骤
                        3. 经典动态规划问题
                        4. 2025年高频面试题

                        动态规划基础

                        核心思想

                        定义：动态规划通过保存子问题的解，避免重复计算，从而提高效率。

                        关键特征：

                        1. 最优子结构：问题的最优解包含子问题的最优解
                        2. 重叠子问题：子问题被多次计算
                        3. 无后效性：一旦某个状态确定，就不受之后决策的影响

                        与其他算法的区别

                        算法特点时间复杂度典型问题
                        分治法子问题独立O(n log n)归并排序、快速排序
                        贪心算法局部最优O(n log n)活动选择、哈夫曼编码
                        动态规划子问题重叠O(n²)或O(n³)背包问题、最长公共子序列

                        动态规划 vs 递归

                        递归（带备忘录）：自顶向下，从大问题分解到小问题
                        动态规划：自底向上，从小问题逐步构建大问题

                        示例：斐波那契数列

                        # 递归（无优化）- 时间复杂度O(2^n）
def fib(n):
    if n <= 1:
        return n
    return fib(n - 1) + fib(n - 2)

# 递归（带备忘录）- 时间复杂度O(n)
def fib_memo(n, memo={}):
    if n in memo:
        return memo[n]
    if n <= 1:
        return n
    memo[n] = fib_memo(n - 1, memo) + fib_memo(n - 2, memo)
    return memo[n]

# 动态规划 - 时间复杂度O(n)
def fib_dp(n):
    if n <= 1:
        return n
    dp = [0, 1]
    for i in range(2, n + 1):
        dp.append(dp[i - 1] + dp[i - 2])
    return dp[n]

# 动态规划（空间优化）- 时间复杂度O(n)，空间复杂度O(1)
def fib_optimized(n):
    if n <= 1:
        return n
    a, b = 0, 1
    for _ in range(2, n + 1):
        a, b = b, a + b
    return b

                        动态规划解题步骤

                        四步法

                        1. 定义状态

                        明确dp数组的含义，dp[i]表示什么。

                        示例：爬楼梯问题

                        • 状态定义：dp[i]表示爬到第i层的方法数

                        2. 状态转移方程

                        找出dp[i]与dp[i-1]、dp[i-2]等的关系。

                        示例：爬楼梯问题

                        • 状态转移：dp[i] = dp[i-1] + dp[i-2]
                        • 含义：到第i层可以从第i-1层爬1步，或从第i-2层爬2步

                        3. 初始化

                        确定初始条件和边界情况。

                        示例：爬楼梯问题

                        • dp[0] = 1（第0层有1种方法：不爬）
                        • dp[1] = 1（第1层有1种方法：爬1步）

                        4. 计算顺序

                        确定是自底向上还是自顶向下计算。

                        示例：爬楼梯问题

                        • 从dp[2]开始，依次计算到dp[n]

                        完整示例：爬楼梯问题

                        题目：假设你正在爬楼梯。需要 n 阶你才能到达楼顶。每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

                        代码实现：

                        def climbStairs(n):
    if n <= 2:
        return n
    
    # 1. 定义状态
    dp = [0] * (n + 1)
    
    # 3. 初始化
    dp[0] = 1
    dp[1] = 1
    
    # 4. 计算顺序（自底向上）
    for i in range(2, n + 1):
        # 2. 状态转移方程
        dp[i] = dp[i - 1] + dp[i - 2]
    
    return dp[n]

# 空间优化版本
def climbStairs_optimized(n):
    if n <= 2:
        return n
    
    a, b = 1, 1
    for _ in range(2, n + 1):
        a, b = b, a + b
    
    return b

                        经典动态规划问题

                        1. 背包问题

                        0-1背包问题

                        题目：给定n个物品，每个物品有重量和价值，背包容量为W，求能装入的最大价值。

                        状态定义：dp[i][w]表示前i个物品，容量为w时的最大价值

                        状态转移：

                        dp[i][w] = max(dp[i-1][w], dp[i-1][w-weight[i]] + value[i])

                        代码实现：

                        def knapsack_01(weights, values, W):
    n = len(weights)
    dp = [[0] * (W + 1) for _ in range(n + 1)]
    
    for i in range(1, n + 1):
        for w in range(1, W + 1):
            # 不选第i个物品
            dp[i][w] = dp[i - 1][w]
            
            # 选第i个物品（如果可以）
            if w >= weights[i - 1]:
                dp[i][w] = max(dp[i][w], 
                              dp[i - 1][w - weights[i - 1]] + values[i - 1])
    
    return dp[n][W]

# 空间优化版本
def knapsack_01_optimized(weights, values, W):
    n = len(weights)
    dp = [0] * (W + 1)
    
    for i in range(n):
        for w in range(W, weights[i] - 1, -1):
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    
    return dp[W]

                        完全背包问题

                        题目：每个物品可以选取多次。

                        状态转移：

                        dp[i][w] = max(dp[i-1][w], dp[i][w-weight[i]] + value[i])

                        代码实现：

                        def knapsack_complete(weights, values, W):
    n = len(weights)
    dp = [0] * (W + 1)
    
    for i in range(n):
        for w in range(weights[i], W + 1):
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    
    return dp[W]

                        2. 最长公共子序列（LCS）

                        题目：给定两个字符串text1和text2，返回它们的最长公共子序列的长度。

                        状态定义：dp[i][j]表示text1的前i个字符和text2的前j个字符的最长公共子序列长度

                        状态转移：

                        如果text1[i-1] == text2[j-1]:
    dp[i][j] = dp[i-1][j-1] + 1
否则:
    dp[i][j] = max(dp[i-1][j], dp[i][j-1])

                        代码实现：

                        def longestCommonSubsequence(text1, text2):
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i - 1] == text2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    
    return dp[m][n]

                        3. 最长递增子序列（LIS）

                        题目：给定一个整数数组nums，找到其中最长严格递增子序列的长度。

                        状态定义：dp[i]表示以nums[i]结尾的最长递增子序列长度

                        状态转移：

                        dp[i] = max(dp[j] + 1) for j < i and nums[j] < nums[i]

                        代码实现：

                        def lengthOfLIS(nums):
    if not nums:
        return 0
    
    n = len(nums)
    dp = [1] * n
    
    for i in range(1, n):
        for j in range(i):
            if nums[j] < nums[i]:
                dp[i] = max(dp[i], dp[j] + 1)
    
    return max(dp)

# 优化版本：二分查找 - O(n log n)
def lengthOfLIS_optimized(nums):
    if not nums:
        return 0
    
    tails = []
    
    for num in nums:
        # 二分查找
        left, right = 0, len(tails)
        while left < right:
            mid = (left + right) // 2
            if tails[mid] < num:
                left = mid + 1
            else:
                right = mid
        
        if left == len(tails):
            tails.append(num)
        else:
            tails[left] = num
    
    return len(tails)

                        4. 编辑距离

                        题目：给定两个单词word1和word2，计算出将word1转换成word2所使用的最少操作数。

                        操作：插入、删除、替换

                        状态定义：dp[i][j]表示word1的前i个字符转换成word2的前j个字符的最少操作数

                        状态转移：

                        如果word1[i-1] == word2[j-1]:
    dp[i][j] = dp[i-1][j-1]
否则:
    dp[i][j] = min(dp[i-1][j] + 1,      # 删除
                  dp[i][j-1] + 1,      # 插入
                  dp[i-1][j-1] + 1)    # 替换

                        代码实现：

                        def minDistance(word1, word2):
    m, n = len(word1), len(word2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    # 初始化
    for i in range(m + 1):
        dp[i][0] = i
    for j in range(n + 1):
        dp[0][j] = j
    
    # 填充dp表
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if word1[i - 1] == word2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1]
            else:
                dp[i][j] = min(dp[i - 1][j] + 1,      # 删除
                              dp[i][j - 1] + 1,      # 插入
                              dp[i - 1][j - 1] + 1)    # 替换
    
    return dp[m][n]

                        5. 零钱兑换

                        题目：给定不同面额的硬币和一个总金额，计算可以凑成总金额所需的硬币的最少个数。

                        状态定义：dp[i]表示凑成金额i所需的最少硬币数

                        状态转移：

                        dp[i] = min(dp[i - coin] + 1) for coin in coins if coin <= i

                        代码实现：

                        def coinChange(coins, amount):
    if amount == 0:
        return 0
    
    # 初始化为amount + 1（表示不可能）
    dp = [amount + 1] * (amount + 1)
    dp[0] = 0
    
    for i in range(1, amount + 1):
        for coin in coins:
            if coin <= i:
                dp[i] = min(dp[i], dp[i - coin] + 1)
    
    return dp[amount] if dp[amount] != amount + 1 else -1

                        高频面试题

                        题目1：不同路径

                        题目描述：
                        一个机器人位于一个 m x n 网格的左上角，机器人每次只能向下或向右移动一步。机器人试图达到网格的右下角，问总共有多少条不同的路径？

                        示例：

                        输入：m = 3, n = 7
输出：28

                        解题思路

                        方法：动态规划

                        状态定义：dp[i][j]表示到达位置(i, j)的路径数

                        状态转移：

                        dp[i][j] = dp[i-1][j] + dp[i][j-1]

                        代码实现

                        Python：

                        def uniquePaths(m, n):
    # 初始化第一行和第一列
    dp = [[1] * n for _ in range(m)]
    
    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = dp[i - 1][j] + dp[i][j - 1]
    
    return dp[m - 1][n - 1]

# 空间优化版本
def uniquePaths_optimized(m, n):
    dp = [1] * n
    
    for i in range(1, m):
        for j in range(1, n):
            dp[j] += dp[j - 1]
    
    return dp[n - 1]

                        复杂度分析

                        • 时间复杂度：O(m * n)
                        • 空间复杂度：O(n) - 优化后

                        题目2：最小路径和

                        题目描述：
                        给定一个包含非负整数的 m x n 网格，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。每次只能向下或向右移动一步。

                        示例：

                        输入：grid = [[1,3,1],[1,5,1],[4,2,1]]
输出：7
解释：路径 1→3→1→1→1 的总和最小。

                        解题思路

                        方法：动态规划

                        状态定义：dp[i][j]表示到达位置(i, j)的最小路径和

                        状态转移：

                        dp[i][j] = grid[i][j] + min(dp[i-1][j], dp[i][j-1])

                        代码实现

                        Python：

                        def minPathSum(grid):
    m, n = len(grid), len(grid[0])
    dp = [[0] * n for _ in range(m)]
    
    # 初始化
    dp[0][0] = grid[0][0]
    for i in range(1, m):
        dp[i][0] = dp[i - 1][0] + grid[i][0]
    for j in range(1, n):
        dp[0][j] = dp[0][j - 1] + grid[0][j]
    
    # 填充dp表
    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = grid[i][j] + min(dp[i - 1][j], dp[i][j - 1])
    
    return dp[m - 1][n - 1]

# 空间优化版本
def minPathSum_optimized(grid):
    m, n = len(grid), len(grid[0])
    dp = [0] * n
    
    dp[0] = grid[0][0]
    for j in range(1, n):
        dp[j] = dp[j - 1] + grid[0][j]
    
    for i in range(1, m):
        dp[0] += grid[i][0]
        for j in range(1, n):
            dp[j] = grid[i][j] + min(dp[j], dp[j - 1])
    
    return dp[n - 1]

                        复杂度分析

                        • 时间复杂度：O(m * n)
                        • 空间复杂度：O(n) - 优化后

                        题目3：打家劫舍

                        题目描述：
                        你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。给定一个代表每个房屋存放金额的非负整数数组，计算你不触动警报装置的情况下，一夜之内能够偷窃到的最高金额。

                        示例：

                        输入：[2,7,9,3,1]
输出：12
解释：偷窃第1、3、5号房屋，金额为2+9+1=12。

                        解题思路

                        方法：动态规划

                        状态定义：dp[i]表示前i个房屋能偷窃到的最高金额

                        状态转移：

                        dp[i] = max(dp[i-1], dp[i-2] + nums[i-1])

                        代码实现

                        Python：

                        def rob(nums):
    if not nums:
        return 0
    if len(nums) == 1:
        return nums[0]
    
    dp = [0] * len(nums)
    dp[0] = nums[0]
    dp[1] = max(nums[0], nums[1])
    
    for i in range(2, len(nums)):
        dp[i] = max(dp[i - 1], dp[i - 2] + nums[i])
    
    return dp[-1]

# 空间优化版本
def rob_optimized(nums):
    if not nums:
        return 0
    if len(nums) == 1:
        return nums[0]
    
    prev2, prev1 = nums[0], max(nums[0], nums[1])
    
    for i in range(2, len(nums)):
        current = max(prev1, prev2 + nums[i])
        prev2, prev1 = prev1, current
    
    return prev1

                        复杂度分析

                        • 时间复杂度：O(n)
                        • 空间复杂度：O(1) - 优化后

                        动态规划优化技巧

                        1. 空间优化

                        原理：如果dp[i]只依赖于dp[i-1]和dp[i-2]，可以用滚动数组优化。

                        示例：斐波那契数列

                        # 未优化
def fib(n):
    dp = [0, 1]
    for i in range(2, n + 1):
        dp.append(dp[i - 1] + dp[i - 2])
    return dp[n]

# 优化后
def fib_optimized(n):
    if n <= 1:
        return n
    a, b = 0, 1
    for _ in range(2, n + 1):
        a, b = b, a + b
    return b

                        2. 状态压缩

                        原理：将二维dp数组压缩为一维数组。

                        示例：0-1背包问题

                        # 未优化
def knapsack(weights, values, W):
    n = len(weights)
    dp = [[0] * (W + 1) for _ in range(n + 1)]
    for i in range(1, n + 1):
        for w in range(1, W + 1):
            if w >= weights[i - 1]:
                dp[i][w] = max(dp[i - 1][w], dp[i - 1][w - weights[i - 1]] + values[i - 1])
    return dp[n][W]

# 优化后
def knapsack_optimized(weights, values, W):
    n = len(weights)
    dp = [0] * (W + 1)
    for i in range(n):
        for w in range(W, weights[i] - 1, -1):
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    return dp[W]

                        3. 路径记录

                        原理：不仅记录最优值，还记录路径。

                        示例：最长公共子序列

                        def lcs_with_path(text1, text2):
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    # 填充dp表
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i - 1] == text2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    
    # 回溯路径
    i, j = m, n
    lcs = []
    while i > 0 and j > 0:
        if text1[i - 1] == text2[j - 1]:
            lcs.append(text1[i - 1])
            i -= 1
            j -= 1
        elif dp[i - 1][j] > dp[i][j - 1]:
            i -= 1
        else:
            j -= 1
    
    return ''.join(reversed(lcs))
