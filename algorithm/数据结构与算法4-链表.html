                                                                                                                                                                          # 前言

                                                                                                                                                                          链表是一种动态数据结构，它通过指针将一系列节点连接起来。与数组不同，链表的元素在内存中不需要连续存储，这使得链表在插入和删除操作上具有天然优势。

                                                                                                                                                                          据统计，LeetCode中约有15%的题目涉及链表操作，是面试中的高频考点。

                                                                                                                                                                          在本文中，我们将深入探讨：

                                                                                                                                                                          1. 链表的核心概念和内存模型
                                                                                                                                                                          2. 单向链表、双向链表、循环链表
                                                                                                                                                                          3. 链表的基本操作和复杂度分析
                                                                                                                                                                          4. 2025年高频面试题
                                                                                                                                                                          5. 快慢指针、虚拟头节点等核心技巧

                                                                                                                                                                          # 链表的本质

                                                                                                                                                                          # 链表 vs 数组

                                                                                                                                                                          特性数组链表
                                                                                                                                                                          内存存储连续不连续
                                                                                                                                                                          访问元素O(1)O(n)
                                                                                                                                                                          插入/删除O(n)O(1)
                                                                                                                                                                          空间大小固定动态
                                                                                                                                                                          缓存友好高低

                                                                                                                                                                          核心区别：

                                                                                                                                                                          • 数组：通过索引直接访问，适合读多写少
                                                                                                                                                                          • 链表：通过指针遍历访问，适合写多读少

                                                                                                                                                                          # 链表的内存模型

                                                                                                                                                                          单向链表内存布局：

地址:  1000      2000      3000      4000
节点:  [A|2000] -> [B|3000] -> [C|4000] -> [D|null]
       数据|next   数据|next   数据|next   数据|next

特点：
- 每个节点包含数据和指向下一个节点的指针
- 节点在内存中不连续
- 最后一个节点的next指针为null

                                                                                                                                                                          # 链表的类型

                                                                                                                                                                          # 1. 单向链表

                                                                                                                                                                          定义：每个节点只包含指向下一个节点的指针

                                                                                                                                                                          结构：

                                                                                                                                                                          class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

                                                                                                                                                                          示例：

                                                                                                                                                                          1 -> 2 -> 3 -> 4 -> None

                                                                                                                                                                          # 2. 双向链表

                                                                                                                                                                          定义：每个节点包含指向前一个和后一个节点的指针

                                                                                                                                                                          结构：

                                                                                                                                                                          class DoublyListNode:
    def __init__(self, val=0, prev=None, next=None):
        self.val = val
        self.prev = prev
        self.next = next

                                                                                                                                                                          示例：

                                                                                                                                                                          None <- 1 <-> 2 <-> 3 <-> 4 -> None

                                                                                                                                                                          优势：

                                                                                                                                                                          • 可以双向遍历
                                                                                                                                                                          • 删除节点更方便（不需要前驱节点）

                                                                                                                                                                          # 3. 循环链表

                                                                                                                                                                          定义：最后一个节点的next指针指向头节点

                                                                                                                                                                          示例：

                                                                                                                                                                          单向循环链表：
1 -> 2 -> 3 -> 4 -> 1 (循环)

双向循环链表：
None <- 1 <-> 2 <-> 3 <-> 4 -> 1 (循环)
           ^                    |
           |____________________|

                                                                                                                                                                          应用：

                                                                                                                                                                          • 轮询调度
                                                                                                                                                                          • 约瑟夫问题
                                                                                                                                                                          • 缓存淘汰策略

                                                                                                                                                                          # 链表的基本操作

                                                                                                                                                                          # 1. 创建链表

                                                                                                                                                                          def create_linked_list(values):
    if not values:
        return None
    
    head = ListNode(values[0])
    current = head
    
    for val in values[1:]:
        current.next = ListNode(val)
        current = current.next
    
    return head

                                                                                                                                                                          # 2. 遍历链表

                                                                                                                                                                          def traverse_linked_list(head):
    current = head
    while current:
        print(current.val)
        current = current.next

                                                                                                                                                                          # 3. 插入节点

                                                                                                                                                                          # 在头部插入

                                                                                                                                                                          def insert_at_head(head, val):
    new_node = ListNode(val)
    new_node.next = head
    return new_node

                                                                                                                                                                          # 在尾部插入

                                                                                                                                                                          def insert_at_tail(head, val):
    new_node = ListNode(val)
    
    if not head:
        return new_node
    
    current = head
    while current.next:
        current = current.next
    
    current.next = new_node
    return head

                                                                                                                                                                          # 在指定位置插入

                                                                                                                                                                          def insert_at_position(head, val, position):
    if position == 0:
        return insert_at_head(head, val)
    
    new_node = ListNode(val)
    current = head
    
    for _ in range(position - 1):
        if not current:
            return head
        current = current.next
    
    new_node.next = current.next
    current.next = new_node
    
    return head

                                                                                                                                                                          # 4. 删除节点

                                                                                                                                                                          # 删除指定值的节点

                                                                                                                                                                          def delete_node(head, val):
    # 使用虚拟头节点简化操作
    dummy = ListNode(0)
    dummy.next = head
    current = dummy
    
    while current.next:
        if current.next.val == val:
            current.next = current.next.next
        else:
            current = current.next
    
    return dummy.next

                                                                                                                                                                          # 删除指定位置的节点

                                                                                                                                                                          def delete_at_position(head, position):
    if position == 0:
        return head.next if head else None
    
    current = head
    for _ in range(position - 1):
        if not current or not current.next:
            return head
        current = current.next
    
    if current.next:
        current.next = current.next.next
    
    return head

                                                                                                                                                                          # 5. 反转链表

                                                                                                                                                                          def reverse_linked_list(head):
    prev = None
    current = head
    
    while current:
        next_node = current.next
        current.next = prev
        prev = current
        current = next_node
    
    return prev

                                                                                                                                                                          # 操作复杂度总结

                                                                                                                                                                          操作单向链表双向链表说明
                                                                                                                                                                          访问头部O(1)O(1)直接返回head
                                                                                                                                                                          访问尾部O(n)O(1)双向链表有tail指针
                                                                                                                                                                          访问中间O(n)O(n)需要遍历
                                                                                                                                                                          头部插入O(1)O(1)直接操作head
                                                                                                                                                                          尾部插入O(n)O(1)双向链表有tail指针
                                                                                                                                                                          中间插入O(n)O(n)需要先找到位置
                                                                                                                                                                          头部删除O(1)O(1)直接操作head
                                                                                                                                                                          尾部删除O(n)O(1)双向链表有tail指针
                                                                                                                                                                          中间删除O(n)O(n)需要先找到位置

                                                                                                                                                                          # 高频面试题

                                                                                                                                                                          # 题目1：反转链表

                                                                                                                                                                          题目描述：
                                                                                                                                                                          给你单链表的头节点 head ，请你反转链表，并返回反转后的链表。

                                                                                                                                                                          示例：

                                                                                                                                                                          输入：head = [1,2,3,4,5]
输出：[5,4,3,2,1]

                                                                                                                                                                          # 解题思路

                                                                                                                                                                          方法1：迭代法

                                                                                                                                                                          1. 使用三个指针：prev、current、next
                                                                                                                                                                          2. 遍历链表，逐个反转指针
                                                                                                                                                                          3. 最后返回prev

                                                                                                                                                                          步骤：

                                                                                                                                                                          原始: 1 -> 2 -> 3 -> None

步骤1:
prev=None, current=1, next=2
1 -> None

步骤2:
prev=1, current=2, next=3
2 -> 1 -> None

步骤3:
prev=2, current=3, next=None
3 -> 2 -> 1 -> None

                                                                                                                                                                          # 代码实现

                                                                                                                                                                          Python：

                                                                                                                                                                          def reverseList(head):
    prev = None
    current = head
    
    while current:
        next_node = current.next
        current.next = prev
        prev = current
        current = next_node
    
    return prev

                                                                                                                                                                          JavaScript：

                                                                                                                                                                          function reverseList(head) {
    let prev = null;
    let current = head;
    
    while (current) {
        const next = current.next;
        current.next = prev;
        prev = current;
        current = next;
    }
    
    return prev;
}

                                                                                                                                                                          # 复杂度分析

                                                                                                                                                                          • 时间复杂度：O(n) - 遍历一次链表
                                                                                                                                                                          • 空间复杂度：O(1) - 只使用常数空间

                                                                                                                                                                          # 题目2：合并两个有序链表

                                                                                                                                                                          题目描述：
                                                                                                                                                                          将两个升序链表合并为一个新的升序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

                                                                                                                                                                          示例：

                                                                                                                                                                          输入：l1 = [1,2,4], l2 = [1,3,4]
输出：[1,1,2,3,4,4]

                                                                                                                                                                          # 解题思路

                                                                                                                                                                          方法1：迭代法

                                                                                                                                                                          1. 使用虚拟头节点简化操作
                                                                                                                                                                          2. 比较两个链表的节点值
                                                                                                                                                                          3. 将较小的节点连接到结果链表
                                                                                                                                                                          4. 处理剩余节点

                                                                                                                                                                          # 代码实现

                                                                                                                                                                          Python：

                                                                                                                                                                          def mergeTwoLists(l1, l2):
    dummy = ListNode(0)
    current = dummy
    
    while l1 and l2:
        if l1.val <= l2.val:
            current.next = l1
            l1 = l1.next
        else:
            current.next = l2
            l2 = l2.next
        current = current.next
    
    # 连接剩余节点
    current.next = l1 if l1 else l2
    
    return dummy.next

                                                                                                                                                                          JavaScript：

                                                                                                                                                                          function mergeTwoLists(l1, l2) {
    const dummy = new ListNode(0);
    let current = dummy;
    
    while (l1 && l2) {
        if (l1.val <= l2.val) {
            current.next = l1;
            l1 = l1.next;
        } else {
            current.next = l2;
            l2 = l2.next;
        }
        current = current.next;
    }
    
    // 连接剩余节点
    current.next = l1 || l2;
    
    return dummy.next;
}

                                                                                                                                                                          # 复杂度分析

                                                                                                                                                                          • 时间复杂度：O(m + n) - m和n分别为两个链表的长度
                                                                                                                                                                          • 空间复杂度：O(1) - 只使用常数空间

                                                                                                                                                                          # 题目3：环形链表

                                                                                                                                                                          题目描述：
                                                                                                                                                                          给你一个链表的头节点 head ，判断链表中是否有环。如果链表中有某个节点，可以通过连续跟踪 next 指针再次到达，则链表中存在环。

                                                                                                                                                                          示例：

                                                                                                                                                                          输入：head = [3,2,0,-4], pos = 1
输出：true
解释：链表中有一个环，其尾部连接到第二个节点。

                                                                                                                                                                          # 解题思路

                                                                                                                                                                          方法1：哈希表

                                                                                                                                                                          1. 遍历链表
                                                                                                                                                                          2. 将访问过的节点存入哈希表
                                                                                                                                                                          3. 如果节点已在哈希表中，说明有环

                                                                                                                                                                          方法2：快慢指针（Floyd判圈算法）

                                                                                                                                                                          1. 使用两个指针：慢指针每次走一步，快指针每次走两步
                                                                                                                                                                          2. 如果有环，快指针终会追上慢指针
                                                                                                                                                                          3. 如果无环，快指针会先到达终点

                                                                                                                                                                          # 代码实现

                                                                                                                                                                          Python（快慢指针）：

                                                                                                                                                                          def hasCycle(head):
    if not head or not head.next:
        return False
    
    slow = head
    fast = head.next
    
    while slow != fast:
        if not fast or not fast.next:
            return False
        slow = slow.next
        fast = fast.next.next
    
    return True

                                                                                                                                                                          JavaScript（快慢指针）：

                                                                                                                                                                          function hasCycle(head) {
    if (!head || !head.next) {
        return false;
    }
    
    let slow = head;
    let fast = head.next;
    
    while (slow !== fast) {
        if (!fast || !fast.next) {
            return false;
        }
        slow = slow.next;
        fast = fast.next.next;
    }
    
    return true;
}

                                                                                                                                                                          # 复杂度分析

                                                                                                                                                                          快慢指针法：

                                                                                                                                                                          • 时间复杂度：O(n) - 最坏情况遍历整个链表
                                                                                                                                                                          • 空间复杂度：O(1) - 只使用常数空间

                                                                                                                                                                          # 题目4：删除链表的倒数第N个节点

                                                                                                                                                                          题目描述：
                                                                                                                                                                          给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

                                                                                                                                                                          示例：

                                                                                                                                                                          输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]

                                                                                                                                                                          # 解题思路

                                                                                                                                                                          方法1：两次遍历

                                                                                                                                                                          1. 第一次遍历：计算链表长度
                                                                                                                                                                          2. 计算要删除的节点位置
                                                                                                                                                                          3. 第二次遍历：删除节点

                                                                                                                                                                          方法2：快慢指针（一次遍历）

                                                                                                                                                                          1. 使用虚拟头节点简化操作
                                                                                                                                                                          2. 快指针先走n步
                                                                                                                                                                          3. 慢指针和快指针同时走，直到快指针到达末尾
                                                                                                                                                                          4. 删除慢指针的下一个节点

                                                                                                                                                                          # 代码实现

                                                                                                                                                                          Python（快慢指针）：

                                                                                                                                                                          def removeNthFromEnd(head, n):
    dummy = ListNode(0)
    dummy.next = head
    
    fast = dummy
    slow = dummy
    
    # 快指针先走n+1步
    for _ in range(n + 1):
        fast = fast.next
    
    # 同时移动，直到fast到达末尾
    while fast:
        slow = slow.next
        fast = fast.next
    
    # 删除slow的下一个节点
    slow.next = slow.next.next
    
    return dummy.next

                                                                                                                                                                          JavaScript（快慢指针）：

                                                                                                                                                                          function removeNthFromEnd(head, n) {
    const dummy = new ListNode(0);
    dummy.next = head;
    
    let fast = dummy;
    let slow = dummy;
    
    // 快指针先走n+1步
    for (let i = 0; i <= n; i++) {
        fast = fast.next;
    }
    
    // 同时移动，直到fast到达末尾
    while (fast) {
        slow = slow.next;
        fast = fast.next;
    }
    
    // 删除slow的下一个节点
    slow.next = slow.next.next;
    
    return dummy.next;
}

                                                                                                                                                                          # 核心技巧

                                                                                                                                                                          # 1. 虚拟头节点

                                                                                                                                                                          用途：简化头节点的操作

                                                                                                                                                                          示例：

                                                                                                                                                                          # 不使用虚拟头节点（需要特殊处理）
def delete_node(head, val):
    if head.val == val:
        return head.next
    
    current = head
    while current.next:
        if current.next.val == val:
            current.next = current.next.next
            break
        current = current.next
    
    return head

# 使用虚拟头节点（统一处理）
def delete_node(head, val):
    dummy = ListNode(0)
    dummy.next = head
    current = dummy
    
    while current.next:
        if current.next.val == val:
            current.next = current.next.next
            break
        current = current.next
    
    return dummy.next

                                                                                                                                                                          # 2. 快慢指针

                                                                                                                                                                          用途：

                                                                                                                                                                          • 寻找中间节点
                                                                                                                                                                          • 检测环
                                                                                                                                                                          • 寻找环的入口

                                                                                                                                                                          示例：寻找中间节点

                                                                                                                                                                          def find_middle(head):
    slow = head
    fast = head
    
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    
    return slow

                                                                                                                                                                          # 3. 递归

                                                                                                                                                                          用途：简化链表操作

                                                                                                                                                                          示例：反转链表

                                                                                                                                                                          def reverseList(head):
    if not head or not head.next:
        return head
    
    new_head = reverseList(head.next)
    head.next.next = head
    head.next = None
    
    return new_head

                                                                                                                                                                          # 4. 双指针

                                                                                                                                                                          用途：

                                                                                                                                                                          • 合并有序链表
                                                                                                                                                                          • 寻找相交节点
                                                                                                                                                                          • 回文链表判断

                                                                                                                                                                          示例：寻找相交节点

                                                                                                                                                                          def getIntersectionNode(headA, headB):
    if not headA or not headB:
        return None
    
    pA, pB = headA, headB
    
    while pA != pB:
        pA = pA.next if pA else headB
        pB = pB.next if pB else headA
    
    return pA

                                                                                                                                                                          # 常见陷阱

                                                                                                                                                                          # 1. 空指针异常

                                                                                                                                                                          常见错误：

                                                                                                                                                                          # 错误：未检查空指针
while current.next:
    current = current.next.next  # current.next可能为None

                                                                                                                                                                          正确做法：

                                                                                                                                                                          while current and current.next:
    current = current.next.next

                                                                                                                                                                          # 2. 丢失链表头

                                                                                                                                                                          常见错误：

                                                                                                                                                                          # 错误：直接修改head
def reverseList(head):
    while head:
        # 修改head，导致丢失原链表
        head.next = None

                                                                                                                                                                          正确做法：

                                                                                                                                                                          def reverseList(head):
    prev = None
    current = head
    while current:
        next_node = current.next
        current.next = prev
        prev = current
        current = next_node
    return prev

def create_cycle(head):

                                                                                                                                                                          正确做法：

    # 明确知道需要循环时才创建
    if need_cycle:

                                                                                                                                                                          # 1. LRU缓存

                                                                                                                                                                          class LRUCache:
    def __init__(self, capacity):
        self.capacity = capacity
        self.cache = {}  # key -> node
        self.head = ListNode(0)  # 虚拟头节点
        self.tail = ListNode(0)  # 虚拟尾节点
        self.head.next = self.tail
        self.tail.prev = self.head
    
    def get(self, key):
        if key in self.cache:
            node = self.cache[key]
            self._remove(node)
            self._add(node)
            return node.value
        return -1
    
    def put(self, key, value):
        if key in self.cache:
            self._remove(self.cache[key])
        
        node = ListNode(key, value)
        self._add(node)
        self.cache[key] = node
        
        if len(self.cache) > self.capacity:
            lru = self.head.next
            self._remove(lru)
            del self.cache[lru.key]
    
    def _remove(self, node):
        prev = node.prev
        next_node = node.next
        prev.next = next_node
        next_node.prev = prev
    
    def _add(self, node):
        prev = self.tail.prev
        prev.next = node
        node.prev = prev
        node.next = self.tail
        self.tail.prev = node

                                                                                                                                                                          # 2. 浏览器历史记录

                                                                                                                                                                          class BrowserHistory:
    def __init__(self, homepage):
        self.current = ListNode(homepage)
    
    def visit(self, url):
        new_node = ListNode(url)
        self.current.next = new_node
        new_node.prev = self.current
        self.current = new_node
    
    def back(self, steps):
        for _ in range(steps):
            if self.current.prev:
                self.current = self.current.prev
        return self.current.val
    
    def forward(self, steps):
        for _ in range(steps):
            if self.current.next:
                self.current = self.current.next
        return self.current.val

                                                                                                                                                                          # 3. 音乐播放列表

                                                                                                                                                                          class MusicPlayer:
    def __init__(self):
        self.head = None
        self.current = None
    
    def add_song(self, song):
        new_node = ListNode(song)
        if not self.head:
            self.head = new_node
            self.current = new_node
        else:
            current = self.head
            while current.next:
                current = current.next
            current.next = new_node
    
    def play_next(self):
        if self.current and self.current.next:
            self.current = self.current.next
            return self.current.val
        return None
    
    def play_previous(self):
        if self.current and self.current.prev:
            self.current = self.current.prev
            return self.current.val
        return None
